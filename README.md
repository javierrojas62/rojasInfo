Sitio web de **Rojas Informática** — servicio técnico de PC y notebooks, representantes Hikvision, venta de equipos y desarrollo de software a medida.

**Paleta:** 3 colores.  
**Efecto principal:** Liquid Glass (Apple-style).  
**Animación:** Red neuronal 3D sutil con Three.js.  

## Servicios

- **Servicio Técnico** — Reparación y mantenimiento de PC y notebooks
- **Hikvision** — Cámaras, porteros eléctricos y alarmas
- **Venta de Equipos** — PC, notebooks, componentes y periféricos
- **Software a Medida** — Apps web, sistemas de gestión, APIs

## Subir a GitHub Pages

```bash
git init
git add .
git commit -m "Rojas Informática — sitio web"
git branch -M main
git remote add origin https://github.com/javierrojas62/rojas-informatica.git
git push -u origin main
```

Settings > Pages > Source: main / (root).
